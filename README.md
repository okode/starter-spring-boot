# Starter Spring Boot App

Created with Spring Initializr (https://start.spring.io)

* Gradle Project
* Languaje Java
* Spring Boot version 2.3.2
* Packaging JAR
* Java version 14
* Dependencies
  - Lombok
  - Spring Reactive Web

## Building

```
./gradlew build bootJar
```

## Running

```
./gradlew bootRun
```

## Bumping Gradle Version

```
./gradlew wrapper --gradle-version [VERSION]
```
